This is the page to demonstrate JS and jQuery.

The goal here is to have input from the user be processed by JS/jQuery to produce a table as an output. The design is to have multiple inputs allowed and to have a consistent output type. Here the implementation is to create a table based upon the users input on a drop down list. This input will be in the form of a career choice that will return a table of values of the salary based upon the experience gained in the field. The user can then input the amount of years working at the chosen career to recieve an estimate of how much their earnings could be.
